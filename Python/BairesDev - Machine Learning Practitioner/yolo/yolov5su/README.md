# Tipos de Redes de Deep Learning

Este repositório contém links para diferentes tipos de redes de Deep Learning, categorizados em **Redes de Classificação**, **Redes de Detecção** e **Redes de Segmentação**. Cada categoria possui documentos e teses que ajudam a entender melhor os diferentes modelos e suas aplicações.
